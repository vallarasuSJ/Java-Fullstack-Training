Typescript:

Browser doesn't understand ts. so we have to use js. 
practice on git bash terminal. it is similar to linux.

----------------------------------------

 $node -v                [to check node is installed].
 $ npm install -g typescript            [ to install typescript globally ,-g  means global, -g for letter , --global for word] 
 $tsc -v                 [to check typescript downloaded]  
 $tsc script.ts       [to convert ts to js]
 $npm init -y           [to create json file]
 $npm install -D parcel parcel-bundler  [to add package]

 // add dev in json 
 $npm run dev  [to get a website link]
